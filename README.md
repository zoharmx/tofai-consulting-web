## Installation

### Prerequisites

- Node.js 18.17 or higher
- npm, yarn, or pnpm

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/eduardo-rdz-portfolio.git
   cd eduardo-rdz-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

## Customization

### Change Colors

Edit `tailwind.config.js`:

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        // Adjust primary green scale
      }
    }
  }
}
```

1. Create folder in `app/case-studies/[name]/`
2. Add `page.tsx` with content
